# Portable

A little pocket for the internet things you love. Save, watch, and read later.

## Features

- **Inbox** — Capture links, articles, and media to process later
- **Queue** — Organize content you're ready to consume
- **Library** — Archive everything you've saved

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.

## Tech Stack

- [Next.js 15](https://nextjs.org) — React framework
- [Supabase](https://supabase.com) — Auth & database
- [Tailwind CSS](https://tailwindcss.com) — Styling
- [shadcn/ui](https://ui.shadcn.com) — UI components
